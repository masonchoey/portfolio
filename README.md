# Mason Choey — Portfolio

A personal portfolio site built with **React + Vite**, deployed to **GitHub Pages** via GitHub Actions.

🔗 **Live:** https://masonchoey.github.io/portfolio

## Tech
- React 18 + Vite 6
- Plain CSS (custom dark theme, fully responsive)
- Automated deploy on push to `main` (`.github/workflows/deploy.yml`)

## Develop
```bash
npm install
npm run dev      # local dev server
npm run build    # production build → dist/
npm run preview  # preview the production build
```

## Editing content
All copy (experience, projects, skills, links) lives in [`src/data/content.js`](src/data/content.js).
